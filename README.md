# Picture Sliding Puzzle

A sliding puzzle game implemented in Java, evolving from a console-based application to a web-based application using Spring Boot.

## Overview

The project demonstrates the design of a modular game application with separated game logic, persistence layer, and web interface.

The game integrates backend services for storing scores, comments, and ratings using PostgreSQL.

## Technologies

- Java
- Spring Boot
- JPA
- REST API
- PostgreSQL
- HTML / CSS

## Features

- Console-based gameplay
- Web graphical interface
- User authentication
- Score tracking
- Comments and rating system
